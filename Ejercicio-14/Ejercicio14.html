Cálculo automático de notas de alumnos

Empleo de API File, API Drag and Drop, API Geolocation

Esta aplicación recibe un fichero y a partir de él genera un texto con la nota textual correspondiente de cada alumno con la localización de la corrección

El funcionamiento consiste en arrastrar el fichero de datos de los alumnos mediante la Api Drag and Drop

Mediante la Api File se lee el contenido del archivo, se analiza y se pone la nota correspondiente al alumno

Los ficheros han de tener el formato "Nombre alumno;nota" en cada línea del fichero

Este análisis genera un resultado que incluye además la localización de la corrección, por si eres un profesor que se mueve bastante, recordar donde has corregido los examenes

O por si olvidaste corregir algún examen, que sepas el lugar donde probablemente esté. Para ello tienes que aceptar el acceso a tu ubicación al iniciar la pantalla

Modo de empleo

Arrastra y suelta AQUÍ el archivo de notas del los alumnos con el formato: "Nombre alumno:nota"
